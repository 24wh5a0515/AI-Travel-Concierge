# AI Travel Concierge

## Project Overview
AI Travel Concierge is an intelligent travel planning system that helps users create personalized travel itineraries. The system uses Large Language Models (LLMs) through the Groq API to generate travel plans based on user preferences such as destination, budget, and travel duration.

## Problem Statement
Planning a trip can be time-consuming and difficult because travelers need to research destinations, accommodations, activities, and transportation. Many users want a simple system that can automatically generate a travel plan.

## Proposed Solution
This project develops an AI-powered travel assistant that generates customized travel itineraries. Users enter travel details, and the system generates a structured travel plan including attractions, activities, and tips.
